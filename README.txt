TV Explorer Pro is a web application that allows users to search for TV shows using the TVmaze API. The app displays results dynamically, allows users to save favorites, and stores search history in local storage. It was built as a practice project to improve skills in JavaScript, API handling, and DOM manipulation.

Features
Search for TV shows using an external API
Display search results dynamically
Add and remove favorite shows
Save search history using local storage
Dark mode toggle
Responsive layout for different screen sizes
Loading skeleton for better user experience
Technologies Used

HTML, CSS, JavaScript, Fetch API, LocalStorage

How to Run

Download the project files and open index.html in a web browser. Alternatively, use a live server extension in VS Code.

API Used

TV shows are fetched from the TVmaze API:
https://api.tvmaze.com
